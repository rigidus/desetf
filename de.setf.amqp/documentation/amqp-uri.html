amqp-uri.lisp
This file defines the amqp-uri class to specialize `puri:uri` for use with the 'de.setf.amqp' library.
The prospective, standard format for AMQP uri is described in the 0.10 version spec[1], in addition to which QPID suggests to how to support unserinfo[2]. In addition to these, exchange and queue values are recognized as query parameters. QPID suggests[3] a syntax for binding URLs, but its interpreation of scheme values is suspect. [1]: http://jira.amqp.org/confluence/download/attachments/720900/amqp.0-10.pdf?version=1 [2]: http://qpid.apache.org/url-format-proposal.html [3]: http://qpid.apache.org/bindingurlformat.html
