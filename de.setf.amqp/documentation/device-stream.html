This file defines a stream interface for AMQP channel and connection instances for the 'de.setf.amqp' library.
Stream-based operations are available through the `device-read-content` and `device-write-content` operators in the event that the size of the body datum is indeterminate. One specialization accepts a function object as the body. This is used to implement `with-open-channel` by passing the form body as the continuation.
