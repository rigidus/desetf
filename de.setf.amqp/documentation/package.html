package.lisp
This file defines the packages for the 'de.setf.amqp' library.
Several packages are used - `_` isolates macro symbols - `de.setf.amqp` (`amqp`) : exports the names of protocol classes and operators. - `de.setf.amqp-state` (`amqps`) : exports the channel and connection state names - `de.setf.amqp.utility` (`amqp.u`) : exports operator names which are not part of the protocol specifications, but should be available to an application. - `de.setf.amqp.implementation` (`amqp.i`) : the package for implementation code.
